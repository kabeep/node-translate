  • new default<ValueType>(executor): default<ValueType>

  • Create a promise that can be canceled.

    Can be constructed in the same was as a Promise constructor, but with an appended onCancel parameter in executor. PCancelable is a subclass of Promise.

    Cancelling will reject the promise with CancelError. To avoid that, set onCancel.shouldReject to false.

    Type Parameters

    • ValueType

    Parameters

    • executor: ((resolve: ((value?: ValueType | PromiseLike<ValueType>) => void), reject: ((reason?: unknown) => void), onCancel: OnCancelFunction) => void)

        • Parameters

          • resolve: ((value?: ValueType | PromiseLike<ValueType>) => void)
              • (value?): void

              • Parameters

                • Optionalvalue: ValueType | PromiseLike<ValueType>

                Returns void

          • reject: ((reason?: unknown) => void)
              • (reason?): void

              • Parameters

                • Optionalreason: unknown

                Returns void

          • onCancel: OnCancelFunction

          Returns void

    Returns default<ValueType>

    Example

    import PCancelable from 'p-cancelable';

    const cancelablePromise = new PCancelable((resolve, reject, onCancel) => {
    	const job = new Job();

    	onCancel.shouldReject = false;
    	onCancel(() => {
    		job.stop();
    	});

    	job.on('finish', resolve);
    });

    cancelablePromise.cancel(); // Doesn't throw an error
